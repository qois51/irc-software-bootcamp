# Submissions

Folder ini adalah tempat pengumpulan tugas peserta bootcamp.

Setiap peserta membuat folder sendiri di sini dengan format:

```
submissions/
├── nama_peserta_1/
│   ├── mission.py
│   ├── explanation.md
│   └── screenshots/
├── nama_peserta_2/
│   ├── mission.py
│   ├── explanation.md
│   └── screenshots/
└── ...
```

Detail cara pengumpulan ada di [assignment/README.md](../assignment/README.md).
